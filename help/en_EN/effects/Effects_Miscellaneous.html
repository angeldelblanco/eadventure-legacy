Miscelanea effects

In this group we find some different effects

The effects we can find in this group are the following

  • Launch a macro: macros are a set of default effects created by the designer of the game. This effect launch all the effects defined on the macro.
  • Cancel action: this effect cancel the default actions. Use it to avoid an object being consumed automatically from the scene to execute a "Take" action, or to avoid an object being consumed from the inventory after performing a "give to" action.
  • Effect with probability: allows to choose between the execution of two effects depending on a probability, gave on a percentage. Through this effect we might create random behaviours on the adventures.
  • Wait time: The game action keeps stopped during the specified time. Serves to emphasize certain aspects of the game, for example, used in combination with changes of scene, you can give the user a preview of a scene without being able to interact with the elements.
