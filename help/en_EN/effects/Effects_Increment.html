Increase value effect

 

Through this effect a chosen variable is increased in the given units

Create an increase value effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.


  • Creation effect panel: afterwards the next panel will appear:



                

In the panel we can select the variable among the list of existents variables in the chapter.
We can also create a new variable. For that purpose we just type the name of the new variable on the panel
Besides, after the variable selection panel, there is another field where we can edit the number of units we will increase the selected variable. We should highlight that those units will be add to the value which the variable has in the execution instant of the effect in the engine.
