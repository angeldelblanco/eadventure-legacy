Adaptation: Initial State

The initial state allows us to load one of the scenes of the chapter at the start of the game and to give values to variables and flags without having to fulfill any condition or state of the LMS. The following figure shows the actions available and that will be used once the adaptation profile is selected for the chapter:

  • Initial Scene: The scene where the chapter will start.
  • Game State: The values that will be set on flags and variables. New variables and flags can be added just writing the name. The "init var" value can be incremented/decremented or be given a certain value.
