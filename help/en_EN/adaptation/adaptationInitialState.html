Adaptation Initial State

The initial state allows us to load an initial scene among the scenes of the chapter and to give values to variables and flags without having to fulfill any condition or state of the LMS. Through link the profile to the chapter there will be realized the actions that in the following image are detailed:

  • Initial Scene: The scene where the chapter will beggin.
  • Game State: The values that will be set on flags and vars. New variables and flags can be added just writing the name. The init vars value can be incremented/decremented or be given a certain value.
