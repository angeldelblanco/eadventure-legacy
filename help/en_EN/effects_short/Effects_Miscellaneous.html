Miscelanea effects

In this group we find some different effects

The effects we can find in this group are the following

  • Launch a macro: the macro are no more than set of default effects created by the designer of the game. This effect launch all the effects defined on the macro.


  • Cancel action: estethis effect cancel the default actions. Use it to avoid an object being consumed automatically from the scene to execute a "Take" ction, or to avoid an object being consumed from the inventary after performing a "give to" action.


  • Effect with probability: allows to choose between the execution of two effects depending on a probability, gave on a porcentage. Through this effect we might create random behaviours on the adventures.


  • Wait time: mantiene The game action keeps stopped during the specified time. Serves to emphasize certain aspects of the game, for example, used in combination with changes of scene, you can give the user a preview of a scene without being able to interact with the elements.
