Activate flag Effect

With this effect we activate the chosen flag.

Create an "activate flag" effect:

  • Select the effect: Select the effect in the creation effects panel. Click Ok


  • Creation effects panel: aftewards an edition panel will appear:

    We can choose on it the effect we would like to activate among all the effect created in the chapter.
    In addition we can create a new effect. For that purpose simple type on it the name of the new effect
