IMS-LOM Metadata

  Metadata set compliant with the IMS-LOM standard

  This editor allows you to modify the mandatory elements defined by the standard. The games are exported as Learning Objects of the type "SCORM v12" will have the metadata filled in this editor

If you want to expand these data, the package must be opened with any editor of learning objects (such as Reload Editor), or open the file "manifest.xml" (included in the root folder of the Learning Object) with any IMS-LOM metadata editor.

  It is noteworthy that this is NOT the same standard as IEEE-LOM.

For more information, check out the website http://www.imsproject.org/metadata/ .
