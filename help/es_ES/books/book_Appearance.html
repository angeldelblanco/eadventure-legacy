En la pesta√±a Apariencia podemos configurar el aspecto del libro. Podemos escoger los siguientes elementos:

  • Imagen del fondo del libro: imagen sobre la que se dibujará el contenido del Libro.
  • Imagen normal para la flecha izquierda / derecha: imagen que determina el aspecto de la flecha izquierda / derecha (flechas para cambiar de página) cuándo el ratón no está encima de ellas.
  • Imagen para la flecha izquierda / derecha cuándo el ratón está encima: aspecto para la flecha izquierda / derecha cuándo el ratón está sobre ella.

NOTA: No es necesario definir todas las flechas. Si sólo aportamos las de un lado, las del contrario serán simétricas.
