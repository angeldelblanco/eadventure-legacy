Effects Wait Time

 

Through this effect the action of the game stays stopped during the specified time. Mediante este efecto se mantiene la acción de juego parada durante el tiempo especificado. It serves to emphasize certain aspects of the game, for example, used in combination with changes of scene, you can give the user a previsualization of a scene without being able to do anything with the elements of the same.

Create a Wait Time effects:

  • Select the effect: select the effect in the creation effects panel. Press OK.


  • Creation effect panel: afterwards the next panel will appear:




    On this panel the time the game will stay stopped can be choose.
