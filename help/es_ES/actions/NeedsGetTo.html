Necesita alcanzar objeto

Esta función no tiene sentido para juegos en primera persona. Solo se puede editar y se utiliza en juegos en tercera persona.

Activado

Cuando una acción tiene la casilla activada, quiere decir que el personaje tendrá que poder caminar hasta el objeto para realizar la acción. Si el personaje no puede alanzarlo, porque hay una barrera en medio por ejemplo, entonces la acción no se lanzará.
Además, cuando la casilla esta activa se puede editar la distancia entre el personaje y el objeto. El personaje intetará quedar a dicha distancia del objeto antes de realizar la acción.

Desactivado

Si la casilla esta desactivada, el jugador permanecerá donde está y realizará la acción desde allí (siempre y cuando se cumplan las demás condiciones).
