Student Feedback Configuration

For the assessment profiles it is possible to select if we want show the assessment report to the user as "feedback" at the end of the chapter. If we select this option, we will be able to configure if we want to send the report automatically by e-mail. Notice that for data privacy it is not possible to send the report by e-mail without showing it first to the learner and asking for his/her consent.

In order for the sending of mails to work it is necessary to fill the information of your SMTP e-mail server, whether an account of Gmail, Hotmail, Yahoo, etc. We must press the "Edit SMTP configuration". The first two fields, SMTP Server and SMTP Port must be provided by each e-mail provider. The second ones correspond with the user's name and password that are used to access the e-mail account.
