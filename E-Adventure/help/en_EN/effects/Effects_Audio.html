Effect play sound

This effect reproduces a sound

Create a play sound effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.
  • Creation effect panel: afterwards the next panel will appear:

In the first place we find the posibitily to play the sound on the background. If this option is not activated, no action will take place (launch more effects, moving the player, etc) till the sound is not completely reproduced.

Next to it we find a field to select the audio file. After pressing it, in addition to the posibility of browsing the directories we can play the file we select, so there is no need of leaving the application to hear it. Once the file is selected, the reproduction button will be active. Through this button we can also pre-audite the sound file.

If you want to delete the selection of the selected file, you can choose another, or press the red X below the path of the file.
