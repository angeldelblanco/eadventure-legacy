Effects

 

In order to control the state of the game we use flags, variables and global states. They are used to add some conditions to some elements of the games dependending on the actual value. On this way we define concrete situations in the games, specifying what must appear on the scene, which actions we might be able to p erform, etc. Through the effects we can change the state of the flags and variables, modifying the concrete states in the game. This happen cause the changes on some flags and variables allow changing conditions. We can find those effects on the "Game State" group.

 

Besides, we can find the next effects groups:

 

  • Multimedia: effects which allow us to show multimedia elements, like audio and video.


  • Notification: effects to notify the user information on some different several ways, like a phrase of one of the characters, a book or a conversation.


  • Trigger event: those effects will trigger some events, like a change of scene.


  • Changes on the scene: effects involving any change in the current scene, as the movement of characters or collection of objects..


  • Miscelanea: miscelanea effects.
