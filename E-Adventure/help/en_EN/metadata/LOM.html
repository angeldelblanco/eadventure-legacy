LOM Metadata



  Set of metadata compliant with the LOM standard

  This editor lets you modify the elements which are considered more relevant in order to clasify games created with this platform. The games exported as Learning Objects of type "SCORM2004, IMS CP and CP WEBCT 4" will have the metadata filled in this editor.

  If you want to expand these data, the package must be opened with any editor of learning objects (such the Reload Editor), or open the file "manifest.xml" (included in the root folder of the Learning Object) with a LOM metadata editor .

  For more information, check out the website http://ltsc.ieee.org/wg12/ .
