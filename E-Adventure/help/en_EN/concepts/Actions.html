Actions

Actions define what can be performed over an object, character or active area.

Type of actions

"Examine" "Take" (only objects and active areas) "Use" "Customizable" "Use with" (only objects) "Give to" (only objects) "Talk to..." (only characters)

Conditions

The conditions of the actions indicate when it can be performed.
If the conditions of the actions are not active, the action button will not be drawn (unless it is an action with a default behaviour, see below). This does not meet if the action has alternative effects. In this case the button will always be drawn.

Alternative effects

Default actions

In all the elements appears by default the "Examine" action, which leads the player to say the "Detailed description" of the element. In the objects appear by default the "take/use" action, even if no action is performed, unless the player indicates that the object cannot be taken, In the characters appears by default the "Talk to..." action, but the player will say that is not posible to maintain a conversation with the character" The behaviour of the default actions can be changed by adding an action of the corresponding type.
