Needs to reach an object

This function has no sense for first-person games. It can only be edited and it is used in third-person games.

Activated

When an action has this field activated that means that the character will have to walk to the object in order to perform an operation. If the character can not reach the object (for instance, because there is a barrier) the action will not be performed.

Besides, when the field is active the distance between the character and the object could be edited. The character will try to stay at that distance before performing the action.

Deactivated

If the field is not activated the character will stay with no movement, and it will perform the action form this place (if the other conditions are already true).
