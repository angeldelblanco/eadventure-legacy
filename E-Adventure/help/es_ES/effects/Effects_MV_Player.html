Move Player effect



Through this effect we can move the player to the selected position.

Create a move player effect:

  • Select the effect: Select the effect on the creation effects panel. Press OK.


  • Creation effect panel: afterwards the next panel will appear:






  • In the first place we are allowed to select one scene of the chapter. Like in other effects, the selection of the scene has the only aim of preview the scene, because the effect might not have any relationshop with the scene. The movement of the player will execute in the scene which execute the effect.
    Once selected the scene, we choose the position where the player will move, either fulfilling X and Y coordinates or just by clicking in the position selected in the scene.
