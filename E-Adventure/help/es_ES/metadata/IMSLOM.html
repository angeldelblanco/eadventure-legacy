Metadatos IMS-LOM


   Conjunto de metadatos siguiendo el estándar IMS-LOM

  En este editor se permite modificar los elementos obligatorios según el estándar. Los juegos exportados como Objetos de Aprendizaje del tipo "SCORM v12" llevarán los metadatos rellenados en este editor.

  En caso de querar ampliar estos datos, se deberá abrir el paquete exportado con algún editor de Objetos de Aprendizaje (como Reload Editor), o abrir el fichero "manifest.xml" (incluido en el fichero raíz del Objeto de Aprendizaje) con algún editor de metadatos IMS-LOM.

  Cabe destacar que este NO es el mismo estándar que IEEE-LOM.

  Para mas información al respecto, consultar la página http://www.imsproject.org/metadata/ .
