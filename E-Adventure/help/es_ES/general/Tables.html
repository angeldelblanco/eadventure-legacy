Tablas

Botones

Añadir

Este botón permite añadir un nuevo elemento a la tabla.

Eliminar

Este botón permite eliminar el elemento seleccionado en la tabla.

Duplicar

Este botón permite duplicar el elemento seleccionado en la tabla. No todas las tabla tienen este tipo de botón.

Mover hacia arriba

Este botón permite mover el elemento seleccionado en la tabla hacia arriba en la misma tabla.

Mover hacia abajo

Este botón permite mover el elemento seleccionado en la tabla hacia abajo en la misma tabla.

Edición

Los elementos de la tabla solo se pueden editar cuando estan seleccionados. Tenga en cuenta que muchos cambian al seleccionarlos para permitir distintos tipos de edición.
