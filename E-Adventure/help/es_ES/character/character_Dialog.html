Personajes: Configuración de diálogo

En la pestaña de Configuración de diálogo podemos definir la configuración de los diálogos para el personaje.

Colores de las líneas de diÁlogo del protagonista

En este apartado, definimos el aspecto del diálogo:

  • Color principal: color de las letras.
  • Color de borde: color para el borde de las letras.
  • Si tenemos activo Muestra bocadillos de diálogo
    • Color de fondo de bocadillos: el color para el contenedor del diálogo.
    • Color de borde de bocadillas: color para el borde del contenedor del diálogo.

Voz actual

En esta opción, podemos especificar que queremos que la voz del protagonista sea sintetizada. Esta opción está aún en fase de desarrollo.
